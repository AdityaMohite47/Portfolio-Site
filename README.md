# Portfolio — Aditya Mohite

Personal developer portfolio built with React + Vite. Deployed via GitHub Actions to GitHub Pages.

## Live Site

[adityamohite47.github.io/portfolio](https://adityamohite47.github.io/portfolio/)

## Stack

- **React** + **Vite**
- **react-icons** — GitHub and LinkedIn icons
- **GitHub REST API** — live stats (repos, commits, active since)
- **GitHub Actions** — CI/CD pipeline to GitHub Pages

## Sections

- **Hero** — name, handles, live GitHub stats, resume download
- **About** — brief introduction
- **Skills** — categorized by Languages, Frontend, Backend, Databases, ML, Tools
- **Contact** — email, GitHub, LinkedIn

## Deploy

Pushes to `main` automatically trigger a build and deploy via `.github/workflows/deploy.yml`.
